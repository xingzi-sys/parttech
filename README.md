# PartTech - 智算生态合作商管理系统

## 技术栈

- 后端：Flask + SQLAlchemy + Flask-JWT-Extended
- 数据库：SQLite
- 前端：Vue 3 + Element Plus + Pinia + Vue Router

## 快速启动

### 后端
```bash
cd backend
pip install -r requirements.txt
python seed.py
python run.py
```

### 前端
```bash
cd frontend
npm install
npm run dev
```
